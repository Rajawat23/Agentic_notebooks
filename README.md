## Overview

This repository explores **agentic AI systems** built mainly with **Ollama** and open-source tooling.  
The goal is to show that useful and practical agentic workflows can be built without depending only on paid providers such as OpenAI, Anthropic, or Gemini.

The repository focuses on:

- building agentic workflows with **local LLMs**
- keeping experimentation **low-cost and accessible**
- demonstrating how the **same design patterns** can often be reused across providers
- learning through **hands-on notebooks, examples, and small implementations**

In most cases, the code is designed so that you can switch to another model provider with only small changes.  
If you have a more powerful machine, you can also run larger Ollama models for stronger performance.

Some notebooks may use an external provider when needed. Whenever that happens, it will be clearly mentioned in the notebook.

---

## Why this repository?

Many agentic AI tutorials rely heavily on paid APIs. This repository takes a different approach.

It is built around **Ollama** to demonstrate:

- the power of **small and local language models**
- lower-cost experimentation for learning and prototyping
- practical open-source workflows for agentic systems
- portability across different model backends

This makes the repository useful for both:

- learners who want to understand agentic design patterns
- practitioners who want to experiment without high API costs
